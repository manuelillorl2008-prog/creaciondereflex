creaciondereflex


Este proyecto consiste en el desarrollo de una aplicación web básica utilizando el framework Reflex. La aplicación demuestra el uso de componentes interactivos creados con Python, permitiendo la manipulación dinámica de contenido en pantalla.

Tecnologías utilizadas
Python
Reflex
Visual Studio Code
Git y GitHub

Instalación
Clonar el repositorio:
git clone https://github.com/TUUSUARIO/creaciondereflex.git
Entrar a la carpeta del proyecto:
cd creaciondereflex
Crear entorno virtual:
python -m venv venv
Activar entorno virtual:
.\venv\Scripts\Activate
Instalar dependencias:
pip install -r requirements.txt

Ejecución

Ejecutar la aplicación con:
python -m reflex run

Luego abrir en el navegador:
http://localhost:3000/

Funcionalidad

La aplicación incluye:

Un título principal
Un contador dinámico
Botones interactivos para incrementar y disminuir el valor
